student api with spring boot 



it is a restful spring boot application for managing student data connected to MySQL 



it has these features 



\- Create Student

\- Get All Students

\- Get Student by ID

\- Update Student

\- Delete Student

\- Global Exception Handling







TECH STACK USED : 



Java 17 

Spring Boot 

Spring Data JPA 

Hibernate 

MySQL 

Maven 







PROJECT STRUCTURE :



src/main/java/com/example/Student\_api

├── controller

├── service

│   └── impl

├── repository

├── dto

├── entity

└── exception







IMPORTANT : 



Database configuration is managed using `application.properties`.  

But application.properties has been excluded 







HOW TO RUN : 



1. Clone the repository

2\. Configure MySQL database



Write database connection by your self in the appliaction.properties  and establish database connection
